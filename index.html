React Native Projects




React Native is a cross-platform framework developed by Facebook that uses compiled JavaScript to create apps. The JavaScript is compiled into the phone's native language to increase the speed of the app. Whilst on paper the thought of a framework to make cross-platform apps sounds good, in reality you miss out on some of the native features on each platform. React Native makes it quick to create apps, but they lack the speed of creating the app in it's native language.



For practice in creating apps in React Native, I created a basic stopwatch. The stopwatch features a start/stop button that changes when pressed, and lap timer button. Whilst the app was quite quick to make, the trouble comes when reaching lap 13 and 14. A certain amount of lag enters the app making it less than useful as a functional stopwatch. React Native does not seem to have the raw horsepower of its native counterparts. I will carry on my development in React Native, but I would recommend against using it for any realtime apps.


iOS Projects




iOS apps for the iPhone and iPad have the highest return on investment of any native mobile app. Despite the iOS App Store only selling half as many apps as the Google Play Store, the financial income for developers is approximately double. This equates to each iOS app making 4x the amount of profit when compared to their Android equivalent. iOS users are more likely on average to pay for an app. iOS apps are composed of either the Objective-C or Swift programming languages, the latter of which is the main focus of Apple's efforts. I therefore decided to learn Swift as my first iOS language with the intention of learning Objective-C if it was later required.
